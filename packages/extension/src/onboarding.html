Welcome to Claudback

Comment on your page. Claude reads it.

Claudback lets you pin comments to elements on any web page, a button that's the wrong colour, a layout that breaks, copy that needs a rewrite. Claude reads them through a local MCP server and if you are developing the page locally, makes the changes.

It shines when you're iterating on a local build with Claude Code. Click the things you want changed and say why, no screenshots, no describing where things are. The comment carries the exact element.

Install the server

Claude starts and stops the server itself, you just register it once. Pick your client:

Run this in your terminal:

claude mcp add claudback -- npx -y claudback-mcp

Add this to your claude_desktop_config.json:

{
  "mcpServers": {
    "claudback": {
      "command": "npx",
      "args": ["-y", "claudback-mcp"]
    }
  }
}
Learn more: finding your config file

The easiest way: in Claude Desktop open Settings → Developer → Edit Config — it opens (or creates) the file for you. To find it manually:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json (paste that into the File Explorer address bar)

Save the file, then fully restart Claude Desktop so it picks up the change.

That's it for installation, in the next step you'll ask Claude for a pairing code to connect this extension.

Pair the extension

Ask Claude for a pairing code, copy this prompt into any Claude Code or Claude Desktop chat that has Claudback installed: 

Give me a Claudback pairing code

Then enter the code Claude gives you:

Ask Claude for a pairing code to connect.

You're set. Here's the loop.

  1. Open the page you're working on and click the Claudback Extension icon in Chrome → Enable for this tab (grant the per-site permission when prompted).
  2. Use the floating button to pin comments to elements. Click the thing, say what you want.
  3. Ask Claude: check my claudback comments and make the changes.
  4. Resolved comments clear automatically as Claude works through them.

You can reopen this guide any time from the extension's options page.

Pin it to your toolbar

So Claudback is one click away: click the puzzle-piece icon next to the address bar, find Claudback, and click the pin.
