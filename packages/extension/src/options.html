Claudback pairing

Ask Claude for a pairing code so this extension can talk to your local Claudback collector — copy this prompt into any Claude Code or Claude Desktop chat that has Claudback installed: 

Give me a Claudback pairing code

Codes look like 7XKQ-M3PD, expire after 10 minutes, and work once.

Paste the token manually instead

The long-lived token is generated on first run of the MCP server and stored at ~/.claudback/token (a hidden folder in your home directory).
